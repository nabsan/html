Showing all files on OSX Mavericks

Friday 28, March 2014 in Tips

隠しファイルをすべて表示する

terminal.appを起動し次を入力すること

 >defaults write com.apple.finder AppleShowAllFiles TRUE
 >killall Finder

(後日談)結局は、FALSEに戻して、非表示に戻しました。

xtrafinder を利用するようにしました。
